# Organize Command

## Description

Organize command is a tool that organizes files in a directory into different categories based on their extension, size, or date.

## Usage

```bash
./organize.sh <directory> <mode>
```

## Modes

- `ext`: Organize by extension
- `size`: Organize by size
- `date`: Organize by date

## Examples

```bash
./organize.sh ./test ext
```
